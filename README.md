# Wasap 3.0 - Frontend

## Descripción

Frontend de la aplicación Wasap 3.0. Este componente proporciona la interfaz de usuario y permite la interacción con el sistema mediante el consumo de los servicios expuestos por el Backend For Frontend (BFF).

## Tecnologías

* HTML5
* CSS3
* JavaScript
* Node.js
* npm

## Requisitos

* Node.js
* npm

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm start
```

La aplicación estará disponible en:

```text
http://localhost:3000
```

## Integración

El frontend consume los servicios expuestos por el Backend For Frontend (BFF), el cual centraliza la comunicación con los microservicios de usuarios y mensajes.

Para un funcionamiento correcto deben estar ejecutándose:

* Backend For Frontend (BFF)
* Microservicio de Usuarios
* Microservicio de Mensajes
